# Sales_Performance_Analysis
Interactive sales dashboard and SQL analysis project, implementing Excel and Sql
Project Overview

This project analyzes sales performance data using SQL and Excel to uncover business insights related to revenue trends, regional performance, profitability, and customer behavior.

# Tools Used
SQL
Microsoft Excel
Pivot Tables
Pivot Charts
Dashboarding
# Project Objectives 
Analyze sales trends
Identify top-performing products
Evaluate regional performance
Examine profitability
Build an interactive dashboard
Dashboard Preview


# Key Insights
Certain regions generated consistently higher revenue.
High sales volume did not always correspond to high profitability.
Seasonal fluctuations affected sales performance.
